# 数组、链表、跳表

## 跳表（Skip List）
### 复杂度
* 空间复杂度：O(n)


### 跳表的特点

注意：只能用于元素有序的情况。
跳表 对标的是平衡树和二分查找，是一种 插入/删除/搜索都是 O(log n) 的数据结构，1989 年出现。

* 它最大的优势是原理简单、容易实现、方便扩展、效率更高。因为在一些热门的项目里用来替代平衡树，如 Redis、LevelDB 等。
* 劣势：维护成本较高

### 如何给有序的跳表加速

* 时间复杂度：查询 O（n）

**优化**

1. 添加一级索引
2. 添加二级索引
3. …
4. n 级索引
